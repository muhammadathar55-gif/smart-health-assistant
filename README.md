# Smart Health Awareness Assistant 🏥

### ## Inspiration 💡
The inspiration came from the "Clarity Deficit" in health data. We wanted to build a bridge between complex medical jargon and human understanding using Gemini AI.

### ## What it does 🚀
Our assistant uses Gemini AI to instantly simplify medical jargon and analyze nutrition labels through a multimodal "scan-and-chat" interface. It transforms complex health data into clear, actionable wellness insights.

### ## How we built it 🏗️
Built with a **React/Node.js** stack, using the **Gemini 1.5 Flash API** for multimodal processing and **Firebase** for secure data handling.

### ## Challenges we faced 🚧
Balancing technical accuracy with user safety was our main hurdle. We used strict prompt engineering to ensure the AI remains an educator, not a doctor.

### ## Built With
* **AI:** Google Gemini 1.5 Flash API
* **Frontend:** React Native / Flutter
* **Backend:** Node.js
* **Database:** Firebase
